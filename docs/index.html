Open Source · C# · MIDI

Write music like code

SoundScript is a tiny, deterministic music DSL — type a melody, get a MIDI file. No DAW. No plugins. Just text → MIDI.

View on GitHub Full Documentation 
melody {
    bpm 120
    C4 E4 G4 | C5
}

How it works

DSL script → Tokenizer → Parser → Interpreter → MIDI file

Try it now

  • 1
    Clone & build
    git clone https://github.com/dharangutti/sound-script.git
    cd sound-script && dotnet build
  • 2
    Run the CLI
    dotnet run --project src/SoundScript.Cli -- run examples/melody.ss
  • 3
    Or launch the web demo
    dotnet run --project src/SoundScript.Web
    Edit scripts in the browser and download MIDI instantly — runs entirely client-side.

Examples

melody.ss — notes & BPM

melody {
    bpm 120
    C4 E4 G4 | C5
}

durations.ss — beat lengths

melody {
    bpm 100
    C4 for 2
    E4 for 1
    G4:4
    C5
}

About the Project

SoundScript is a tiny, deterministic music DSL created to turn simple text into MIDI. It is intentionally minimal, hobby‑grade, and designed for clarity, reproducibility, and joy. The language runs entirely through a tokenizer → parser → interpreter → MIDI generator pipeline.
